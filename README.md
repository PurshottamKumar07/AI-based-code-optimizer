# 🚀 AI Code Optimizer

An intelligent **AI-based Code Optimizer** built using Python that detects errors, automatically fixes code, and applies advanced compiler optimization techniques.

---

## 📌 Overview

AI Code Optimizer is a smart desktop application that analyzes source code written in multiple programming languages and improves it by:

* Detecting syntax errors in real-time
* Automatically fixing common coding mistakes
* Applying compiler-level optimizations
* Providing AI-based optimization suggestions

This project combines **Machine Learning + Compiler Design concepts + GUI development** into one powerful tool.

---

## 🔥 Key Features

### ✅ Error Detection

* Detects syntax issues in:

  * C
  * C++
  * Java
  * Python
* Real-time error highlighting

### ⚡ Auto Code Fixing

* Fixes:

  * Missing semicolons
  * Incorrect data types
  * Invalid loop syntax
  * Missing colons (Python)
  * String/char formatting issues

### 🧠 AI-Based Suggestions

* Uses Machine Learning (**Naive Bayes**) to suggest optimization techniques

### 🚀 Code Optimization Techniques

* Constant Folding
* Algebraic Simplification
* Dead Code Elimination
* Copy Propagation
* Common Subexpression Elimination

### 🎨 GUI Interface

* Built with **Tkinter**
* Features:

  * Syntax highlighting
  * Real-time error panel
  * One-click optimization
  * Dark/Light mode

### 🎤 Additional Features

* Voice input support
* PDF code extraction
* Copy & Save output

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/AI-Code-Optimizer.git
cd AI-Code-Optimizer
```

### 2️⃣ Install Dependencies

```bash
pip install scikit-learn pypdf speechrecognition
```

*(Optional for voice feature)*

```bash
pip install pipwin
pipwin install pyaudio
```

### 3️⃣ Run the Application

```bash
python main.py
```

## 🧪 How It Works

1. User inputs code into the editor
2. System detects programming language
3. Errors are identified in real-time
4. Auto-fix module corrects common mistakes
5. Optimization engine applies multiple passes
6. AI model suggests best optimization technique
7. Optimized code is displayed with score

---

## 📊 Optimization Score

The system calculates a score based on:

* Code reduction
* Simplification
* Efficiency improvement
